MS2 to feature grouping

Description

Call this method on any (aligned) feature list to assign all fragmention scans to their corresponding features.

Important note:

This module was also introduced because of a data structure change in MZmine 2.38. All projects that were created with older version should run this module on their feature lists to assign all MS2 scans to their features.

Method parameters

Limit by RT edges
Use the feature's retention time edges to filter MS2 scans. Only scans within the feature's RT range are considered
