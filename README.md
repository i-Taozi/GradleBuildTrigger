What's working:
* Pairing
* Show current pump state
* Show cartridge amount
* Show battery amount
* Show current TBR
* Show active boluses
* Change active basal profile
* Issue boluses
* Issue TBRs
* Cancel boluses
* Cancel TBR
* Start/Stop pump
* Receive and dismiss pump alerts
* Retrieve and display history
* Secure API for other apps like AndroidAPS
* Edit BR profiles
