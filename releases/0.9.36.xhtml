Lilith 0.9.36

Download Now!
(Released: 2009-07-20)

Changes

  • Added missing contact page to help.
  • Added new messageRegex example groovy condition.
  • Updated Groovy to 1.6.3
  • Some locks and blocking queues are fair now.
  • Changed shortcut of "Import" to "command shift I". It collided with "Add condition...", previously.
    This was reported by Scott Stanton.
    (Ticket 60)
  • Implemented Drag and Drop of Lilith files.
    (Ticket 47)
  • Option to disable splash screen.
    This was requested by Thorbjoern Ravn Andersen.
    (Ticket 25)
  • Option to ask before exit.
    This was requested by Lothar Cezanne.
    (Ticket 26)
  • Added "Startup & Shutdown" pane to preferences dialog.
  • Added "Troubleshooting" pane to preferences dialog.
    (Ticket 55)
  • MultiplexSocketAppenderBase is now extending UnsynchronizedAppenderBase instead of AppenderBase.
    This was suggested by Maarten Bosteels and Ralph Goers.
  • Added new option "Show stacktrace of Throwables" so non-technical people can disable it to just see the exception and, if available, message. Clicking on the name or message of an exception will now also open the code in IDEA.
  • "Copy call stack" and "Copy Throwable" are now prepending the StackTraceElements with "\tat " as requested in ticket:61. The detailsView is also displaying "at " before any StackTraceElement.
    (Ticket 61)
  • Added option to color the entire row of a table according to the level/status of the event.
    This was requested by Scott Stanton.
    (Ticket 56)
  • Changes to support Logback 0.9.16. Updated SLF4J to 1.5.8.
    (Ticket 39)
