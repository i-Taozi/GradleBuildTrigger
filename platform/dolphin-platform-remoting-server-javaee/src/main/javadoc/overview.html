This is the documentation of the JavaEE plugin API of Dolphin Platform.

The Dolphin Platform is a framework that implements the presentation model pattern and provides a modern way to create enterprise applications. The Platform provides several client implementations that all can be used in combination with a general sever API. By doing so you can create enterprise application with a single server and several desktop, web and mobile client implementations. Here the Dolphin Platforms define a mechanism to automatically snchronize models between the server and the client.

This module contains the public and general API of the Dolphin Platform JavaFX client API that is used for Java based clients.

The JavaEE plugin of the Dolphin Platform don't contain any public API. All public API can be found in the Dolphin Platform server module.

The JavaEE plugin of the Dolphin Platform is still in development and should currently no be used!

You can find the documentation of the different modules here.

Additional information can be found on the Dolphin Platform website.
