ToC

Keyboard

Notation

^
represents the system dependent command key, e.g. "Ctrl" on Windows and "cmd ⌘" on macOS.
alt
represents "Alt" ("alt ⌥" on Mac).
⇧
represents "Shift".
↵
represents "Enter" or "Return".

Preferences…

Edit

^+C
Copy selection
Copies the selection to the clipboard.
^+D
Go to source
Show source in IDE if Lilith plugin is installed.
^+⇧+V
Paste StackTraceElement
Paste StackTraceElement from clipboard and open code in IDE if Lilith plugin is installed.
^+⇧+C
Copy message
Copies the message of the logging event to the clipboard.
^+⇧+alt+C
Copy message pattern
Copies the message pattern of the logging event to the clipboard.
^+⇧+N
Copy logger name
Copies the logger name of the logging event to the clipboard.
^+⇧+T
Copy Throwable
Copies the Throwable of the logging event to the clipboard.
^+⇧+alt+T
Copy Throwable name
Copies the Throwable class name of the logging event to the clipboard.
^+⇧+S
Copy call location
Copies the call location, i.e. the first element of the logging events call stack, to the clipboard.
^+⇧+alt+S
Copy call stack
Copies the call stack of the logging event to the clipboard.
